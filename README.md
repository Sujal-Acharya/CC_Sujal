# DebtFree — Credit Card Debt Escape Planner

## About
DebtFree helps Indian credit card users understand the true cost of their debt and build a personalized escape plan. 
**100% frontend. No backend. No data storage. No login.**

## Features
- **Debt Clarity**: See exactly how much interest you're paying and how long you'll be trapped if you only pay the minimum.
- **Escape Plan**: Choose between **Avalanche** (highest interest first) or **Snowball** (lowest balance first) strategies.
- **What-If Simulator**: Visualize how small extra payments can save you years of debt and lakhs in interest.
- **100% Private**: All calculations happen in your browser. No data ever leaves your device.

## Tech Stack
- **React (Vite)**
- **Tailwind CSS** (Styling)
- **Framer Motion** (Animations)
- **Recharts** (Interactive Charts)
- **React Router DOM v6** (Navigation)

## Getting Started
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
